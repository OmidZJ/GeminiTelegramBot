# Telegram Bot

This project is a Telegram bot that utilizes generative AI to respond to user messages and generate images.

## Setup

1. Clone the repository.
2. Create a `.env` file and set the following environment variables:
   - `TELEGRAM_BOT_TOKEN`: Your Telegram bot token.
   - `GEMINI_API_KEY`: Your Gemini API key.
3. Run `make install` to install the dependencies.
4. Run `make run` to start the bot.

## Usage

- Send a message or media to the bot.
- Use commands like `/start`, `/help`, `/models`, and `/generate_image <prompt>`.
